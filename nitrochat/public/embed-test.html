🚀 NitroChat Embed Test Page

This page demonstrates how to integrate NitroChat into any website using a simple script tag.

Integration Code

<!-- Add this script tag before closing </body> tag -->
<script src="http://localhost:3003/embed.js" async></script>
Note: The widget automatically loads configuration from the NitroChat server. No client-side configuration needed!

Features

  • ✅ Floating chat button (bottom-right)
  • ✅ Smooth open/close animations
  • ✅ Mobile-responsive (full-screen on mobile)
  • ✅ Server-side configuration
  • ✅ Theme colors from your NitroChat config
  • ✅ Secure iframe isolation

JavaScript API

You can programmatically control the widget:

Try the API:

// Open the chat widget
window.NitroChat.open();

// Close the chat widget
window.NitroChat.close();

// Toggle the chat widget
window.NitroChat.toggle();

Production Setup

For production, update the script URL to your deployed NitroChat instance:

<script src="https://your-nitrochat-domain.com/embed.js" async></script>

Look for the chat button in the bottom-right corner! 👉
